## 기능설명

1. 로그인 & 로그아웃 (Next-auth)
2. Post 등록
3. Post 조회
4. Post 좋아요, 북마크 기능
5. Post Comment 기능
6. 사용자 검색 기능
7. 팔로잉 바(팔로우한 사용자 리스트) 기능
8. 사용자 Follow, Unfollow 기능
9. 사용자 별 게시글 목록 조회
10. 페이지 별 사용자 검증 기능

- 점진적 추가(유지보수) 예정
